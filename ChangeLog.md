# NewLife.Agent 版本更新记录

## v10.16.2026.0702 (2026-07-02)

### Web 管理面板
- **轻量级 Web 管理面板**：新增内置 Web 管理面板，可通过浏览器查看服务状态、执行运维操作，无需登录服务器。
- **扩展面板与主题切换**：支持第三方扩展面板注册，内置多套主题可切换。
- **ApiController 统一 API**：重构 WebPanel 路由体系，新增 ApiController 统一处理 API 请求，规范接口风格。
- **鉴权支持**：Web 面板内置 Token 鉴权机制，保障管理接口安全。

### 文档与协作
- **需求与竞品文档**：新增需求文档与竞品分析报告，完善项目文档体系。
- **依赖库版本升级**：升级 NewLife.Core 至最新版本。

## v10.15.2026.0402 (2026-04-02)

- **重构 Desktop 支持**：支持 Windows 服务拉起桌面应用进程，增强服务下用户进程启动能力。
